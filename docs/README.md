# Makima User Guide

The following user guide is written for Window users. The application may or may not work with other operating systems.
Makima is a desktop app for managing generic tasks. Makima implements the following type of tasks:

1. ToDo: a named task
2. Deadline: a named task with an end date
3. Event: a named task with a start and end date
